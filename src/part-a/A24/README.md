# A24: CSSのみのステッパーUI

以下の要件を満たす、CSS・HTMLのみで動作するステッパー（手順表示）UIを作成しなさい。JavaScriptは利用できない。

- ステップは `STEP 1`・`STEP 2`・`STEP 3`・`STEP 4` の4つを横並びで表示すること
- 各ステップは円形のアイコンと下部のラベルで構成すること
- `:checked` 擬似クラスとチェックボックスを活用し、チェックしたステップをアクティブ状態（色変化）にすること
- アクティブなステップ間を繋ぐ線もアクティブ色に変化させること
- 配色は任意とするが、アクティブ・非アクティブが視覚的に判別できること
