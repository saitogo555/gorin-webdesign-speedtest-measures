# A44: CSS グリッドによるダッシュボードレイアウト

CSS Gridを使用して、以下の仕様のダッシュボード風レイアウトを作成しなさい。

- ページ全体を以下の領域に分割する
  - ヘッダー（header）: 全幅、上部に固定
  - サイドバー（sidebar）: 左側、ヘッダーの下から最下部まで、幅は200px固定
  - メインコンテンツ（main）: サイドバーの右側、残り幅を占める
  - フッター（footer）: 全幅、最下部
- 各領域には背景色を設定し、領域名のテキストを表示して視認できるようにすること
- ページ全体の高さが `100vh` になるよう設定すること
