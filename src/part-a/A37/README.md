# A37: @containerクエリを使ったコンテナ対応カード

以下の要件を満たすコンテナクエリ対応のカードコンポーネントを作成しなさい。JavaScriptは利用できない。

- 親要素に `container-type: inline-size` を設定すること
- 親コンテナが `600px` 以上の場合、カード内の画像とテキストを横並び（`flex-direction: row`）に配置すること
- 親コンテナが `600px` 未満の場合、カード内の画像とテキストを縦並び（`flex-direction: column`）に配置すること
- 少なくとも2つの異なる幅のコンテナにカードを配置し、レイアウトの変化を確認できること
