# A43: CSS のみのトースト通知

CSSのみ（JavaScriptは利用しない）で、以下のようなトースト通知UIを作成しなさい。

- 4種類のトーストを縦並びに表示する
  - 「成功（Success）」「情報（Info）」「警告（Warning）」「エラー（Error）」
- 各トーストは左端に色付きのボーダー（border-left）を持つ
  - 成功: 緑、情報: 青、警告: 黄色、エラー: 赤
- 各トーストは右上に「✕」と表示されたテキストを持つ（クリック機能は不要、デザインのみ）
- ページ読み込み時、各トーストが下からフェードインするアニメーションをCSSで実装する
  - `@keyframes` を使用すること
  - 1番目は0.1s、2番目は0.3s、3番目は0.5s、4番目は0.7sのディレイを設定する
- トースト全体をページ右下に固定表示（`position: fixed`）すること
