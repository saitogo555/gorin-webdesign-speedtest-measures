# B47: リアルタイム文字数カウンター（制限付き）

テキストエリアへの入力に連動してリアルタイムに文字数をカウントする機能を実装しなさい。

- テキストエリアを1つ配置する
- 入力された文字数と最大文字数（140文字）を「12 / 140」のように表示する
- 文字数が 120文字以上 になったら、カウンター表示を黄色にする
- 文字数が 140文字を超えた場合（超過した場合）は、カウンター表示を赤にし、テキストエリアに赤いボーダーを表示する
- 最大文字数を超えている場合、投稿ボタンを `disabled` 状態にする
- 投稿ボタンが有効な状態でクリックされた場合、「投稿しました！」とアラートを表示し、テキストエリアの内容をクリアする
