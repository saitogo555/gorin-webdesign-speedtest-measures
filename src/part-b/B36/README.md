# B36: pagehide/pageshowイベントを使ったページ訪問履歴

以下の要件を満たすページ訪問履歴ツールを作成しなさい。`visibilitychange` イベントは使用せず、**`pagehide` / `pageshow`** を使用すること。

- `pageshow` イベントで「ページが表示されました」+日時をリストに追記すること
- `pagehide` イベントで「ページが離れられました」+日時をリストに追記すること（同期的処理のみ使用すること）
- 履歴は `sessionStorage` に保存し、ページを戻って再表示した際も履歴が維持されること
- 最大10件の履歴を表示し、それ以上は古いものから削除すること
