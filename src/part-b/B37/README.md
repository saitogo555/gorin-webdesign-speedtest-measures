# B37: dragenter/dragleave/dropを使ったファイルドロップゾーン

以下の要件を満たすファイルドロップゾーンを作成しなさい。リスト項目のソートではなく、**ファイルのドロップ**を扱うこと。

- `dragenter` でドロップゾーンの枠線色が変化すること
- `dragleave` でドロップゾーンの枠線色が元に戻ること
- `dragover` のデフォルト動作を `preventDefault()` でキャンセルすること
- `drop` でドロップされたファイルの名前・サイズ・タイプをリスト表示すること
- 画像ファイルがドロップされた場合は `FileReader` でプレビュー表示すること
