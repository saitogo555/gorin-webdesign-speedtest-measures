# B39: errorイベントを使ったグローバルエラーハンドラー

以下の要件を満たすグローバルエラーハンドリング機能を実装しなさい。

- `window.addEventListener('error', ...)` でJavaScriptのランタイムエラーを捕捉すること
- `window.addEventListener('unhandledrejection', ...)` でPromiseの未処理拒否を捕捉すること
- エラーが発生したらページ右下にトースト通知（エラーメッセージ・ファイル名・行番号）を表示すること
- 「エラーを発生させる」ボタン（意図的に例外を投げる）と「Promiseエラーを発生させる」ボタンを設置してテストできること
- トースト通知は5秒後に自動的に非表示になること
