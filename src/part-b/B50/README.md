# B50: ローカルストレージを使ったToDoリスト

ローカルストレージを使い、ページをリロードしても内容が保持されるToDoリストを実装しなさい。

- テキストフィールドと「追加」ボタンを用意する
- 「追加」ボタンをクリックするか、Enterキーを押すとタスクが追加される
- 入力値が空の場合は追加しない（アラートで警告を表示する）
- 各タスクには「削除」ボタンを設け、クリックするとそのタスクが削除される
- 各タスクをクリックすると、取り消し線（`text-decoration: line-through`）でスタイルが変化し、完了状態を表す
- タスクの一覧、各タスクの完了状態はすべて `localStorage` に保存し、ページリロード後も復元される
- 「全削除」ボタンを用意し、クリックするとすべてのタスクと `localStorage` のデータが削除される
