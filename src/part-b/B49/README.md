# B49: 数値入力によるFizzBuzz出力

以下の仕様に従い、入力した数値の範囲でFizzBuzz的なルールを適用して画面に出力するアプリを実装しなさい。

- 「開始値」と「終了値」を入力できる2つの数値入力フィールドを用意する
- 「実行」ボタンをクリックすると、開始値から終了値の範囲で以下のルールに従い出力する
  - 3の倍数 かつ 5の倍数 の場合は「FizzBuzz」と表示（赤色）
  - 3の倍数のみの場合は「Fizz」と表示（青色）
  - 5の倍数のみの場合は「Buzz」と表示（緑色）
  - 7の倍数のみの場合は「Jazz」と表示（紫色）
  - それ以外は数字をそのまま表示
- 開始値が終了値より大きい場合は「開始値は終了値より小さくしてください」とエラー表示する
- 終了値が 200 を超える場合は「200以下で入力してください」とエラー表示する
- 出力はリスト形式（`<ul>` と `<li>`）で表示すること
