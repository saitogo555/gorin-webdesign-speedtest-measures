# B41: pointercancelを含む完全なポインターイベントを使ったタッチ対応スライダー

以下の要件を満たすタッチ対応カルーセルスライダーを作成しなさい。お絵かきツールではなく、**スワイプ操作**に `pointer` イベントを応用すること。

- `pointerdown` でスワイプ開始座標を記録すること
- `pointermove` でスワイプ量に応じてスライダーをリアルタイムで移動させること
- `pointerup` でスワイプ量が閾値以上なら次/前のスライドに切り替え、未満なら元に戻すこと
- `pointercancel` が発生した場合はスライダーを元の位置に戻すこと
- スライドは3枚以上用意し、無限ループすること
