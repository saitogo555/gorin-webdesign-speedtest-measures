# B43: touchstart/touchmove/touchendを使ったスワイプ検知

以下の要件を満たすスワイプ方向検知ツールを作成しなさい。`pointer` イベントではなく、**`touch` イベント**を使用すること。

- `touchstart` でタッチ開始座標を記録すること
- `touchmove` でデフォルトのスクロールを `preventDefault()` でキャンセルすること
- `touchend` で水平・垂直方向のスワイプ距離を計算し、「上」「下」「左」「右」のどの方向にスワイプされたかを判定して表示すること
- スワイプ距離が50px未満の場合は「タップ」と表示すること
