# B30: matchMediaのchangeイベントを使ったブレークポイント監視

以下の要件を満たすブレークポイントモニターを作成しなさい。`resize` イベントは使用せず、**`window.matchMedia().addEventListener('change', ...)` のみ**を使用すること。

- `768px`・`1024px`・`1280px` の3つのブレークポイントを `matchMedia` で監視すること
- 各ブレークポイントを超えた・下回ったタイミングで画面上にメッセージを表示すること
- 現在どのブレークポイント範囲にいるかをラベルで表示すること（例：「mobile」「tablet」「desktop」「wide」）
- ページ読み込み時にも現在のブレークポイントを判定して表示すること
