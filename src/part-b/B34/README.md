# B34: IntersectionObserverを使ったカウントアップアニメーション

以下の要件を満たすカウントアップアニメーションを作成しなさい。スクロールアニメーションではなく、**カウントアップ**に `IntersectionObserver` を応用すること。

- 数字を表示する要素（例：「受注件数」「利用者数」「満足度(%)」）を3つ以上配置すること
- 各要素が画面内に入った時に初めてカウントアップアニメーションが開始されること
- カウントアップは一定時間（例：2秒）でターゲット数値に到達するよう `requestAnimationFrame` を使ってスムーズに動作させること
- 一度カウントアップが完了した要素は再度スクロールして入ってきても再カウントしないこと
