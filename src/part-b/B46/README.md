# B46: キーボード操作による画像スライダー

キーボードの矢印キーで操作できる画像スライダーを実装しなさい。

- 画像の代わりに、番号（1〜5）と背景色が異なるカードを5枚用意する
  - 例: 1枚目は赤、2枚目は青、3枚目は緑、4枚目は黄色、5枚目はオレンジ
- 初期状態で1枚目を表示する
- 右矢印キー（`ArrowRight`）で次のカードへ切り替わる
- 左矢印キー（`ArrowLeft`）で前のカードへ切り替わる
- 最後のカードで右矢印キーを押すと1枚目に戻る
- 最初のカードで左矢印キーを押すと最後のカードへ移動する
- 現在のカード番号を「3 / 5」のように表示する
- カードの切り替え時はCSSトランジションでスムーズに切り替わること
