# C10: 文字列操作関数を使ったテキスト加工ツール

PHP の文字列関数を使ったテキスト加工ツールを作成しなさい。

- テキストエリアに文章を入力できること
- 以下の操作をそれぞれボタンで実行できること
  - 文字数カウント（`mb_strlen()`）
  - 単語数カウント（`str_word_count()`）
  - 大文字・小文字変換（`strtoupper()` / `strtolower()`）
  - 特定の文字列の検索と置換（`str_replace()`）
  - 文字列の逆順（`strrev()`）
- 結果をページ内に表示すること（ページリロードまたはPOST送信で可）
