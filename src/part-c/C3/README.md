# C3: ファイルアップロードと拡張子バリデーション

PHP を使ってファイルアップロード機能を実装しなさい。

- HTMLフォームからファイルをアップロードできること（`enctype="multipart/form-data"`）
- アップロード可能なファイル形式は JPEG・PNG・GIF のみとすること
- ファイルサイズは2MB以下に制限すること
- バリデーション失敗時は適切なエラーメッセージを表示すること
- アップロード成功時はファイル名・サイズ・MIMEタイプを表示すること
- アップロードされたファイルはリネームして `uploads/` ディレクトリに保存すること
