# C4: セッションを使ったログイン機能

PHP のセッション機能を使ったログイン・ログアウト機能を実装しなさい。

- ログインフォームにユーザー名とパスワードの入力欄を用意すること
- ユーザー情報は配列で定義すること（DBは不要）
- ログイン成功時はセッションにユーザー情報を保存し、マイページにリダイレクトすること
- 未ログイン状態でマイページにアクセスするとログインページにリダイレクトすること
- ログアウト時はセッションを破棄すること（`session_destroy()`）
- パスワードは `password_hash()` / `password_verify()` を使用すること
