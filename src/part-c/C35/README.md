# C35: PHPによるソート機能付きデータテーブル

以下のPHP連想配列のデータをHTMLテーブルとして表示し、列ヘッダーをクリックするとソートできる機能を実装しなさい。（※ JavaScriptは利用できない）

```php
$users = [
    ['id' => 3, 'name' => '田中 太郎', 'age' => 28, 'score' => 85],
    ['id' => 1, 'name' => '鈴木 花子', 'age' => 35, 'score' => 92],
    ['id' => 5, 'name' => '佐藤 次郎', 'age' => 22, 'score' => 78],
    ['id' => 2, 'name' => '山田 三郎', 'age' => 41, 'score' => 63],
    ['id' => 4, 'name' => '伊藤 四郎', 'age' => 19, 'score' => 95],
];
```

- テーブルには「ID」「名前」「年齢」「スコア」の4列を表示する
- 各列のヘッダーはリンク（`<a>`タグ）になっており、クリックするとその列で昇順・降順が切り替わるGETパラメータを送信する
  - 例: `?sort=age&order=asc`
- GETパラメータの `sort`（列名）と `order`（`asc` または `desc`）に基づいてPHPの `usort()` で配列を並べ替えて表示する
- 現在ソート中の列ヘッダーには「▲」（昇順）または「▼」（降順）のマークを付ける
- デフォルト（パラメータなし）では `id` の昇順で表示する
- SQLインジェクションに相当するGETパラメータ改ざんを防ぐため、`sort` の値は `['id', 'name', 'age', 'score']` の中にある値のみ受け付けること
