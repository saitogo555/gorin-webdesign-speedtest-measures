# C34: POSTフォームによるBMI計算と結果表示

PHPを使い、フォームから送信された身長・体重をもとにBMIを計算して表示するプログラムを作成しなさい。（※ JavaScriptは利用できない）

- 身長（cm）と体重（kg）を入力できるフォームを作成する（POSTメソッド使用）
- 「計算する」ボタンを押すと、同一ページにBMI値と判定結果を表示する
  - BMI = 体重(kg) ÷ 身長(m)²
  - 判定基準は以下の通り
    - 18.5未満: 「低体重（やせ）」
    - 18.5以上25未満: 「普通体重」
    - 25以上30未満: 「肥満（1度）」
    - 30以上: 「肥満（2度以上）」
- 入力値のバリデーション処理を行うこと
  - 身長・体重が未入力の場合は「入力してください」と表示
  - 身長・体重が数値でない、または0以下の場合は「正しい値を入力してください」と表示
- BMI値は小数点以下第2位まで表示する（例: `22.86`）
- XSS攻撃を防ぐセキュリティ対策（`htmlspecialchars`）を行うこと
